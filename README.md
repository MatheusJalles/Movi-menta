# Movimenta

Aplicação web para organização e gerenciamento de tarefas com criação, edição e exclusão em tempo real.

---

## Tecnologias

- JavaScript (DOM puro)
- HTML5 / CSS3

## Funcionalidades

- Criar, editar e excluir tarefas
- Persistência local com LocalStorage
- Interface responsiva

## Como rodar localmente

```bash
git clone https://github.com/MatheusJalles/Movi-menta
# abrir index.html no navegador
```
