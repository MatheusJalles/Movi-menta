# ✅ Movimenta — Gerenciador de Tarefas

Aplicação web para organização e gerenciamento de tarefas diárias com criação, edição e exclusão em tempo real.

> **Status:** ✅ Concluído

---

## 🎯 Problema que resolve

Uma forma simples e direta de organizar tarefas sem depender de apps complexos ou com excesso de funcionalidades. Foco em clareza e praticidade.

---

## 🚀 Funcionalidades

- ➕ Criar novas tarefas
- ✏️ Editar tarefas existentes
- 🗑️ Excluir tarefas
- 📋 Listar e organizar tarefas por status

---

## 🛠️ Tecnologias

| Camada | Tecnologia |
|--------|-----------|
| Estrutura | HTML5 semântico |
| Estilo | CSS3 |
| Lógica | JavaScript (DOM manipulation) |
| Controle de versão | Git / GitHub |

---

## 🖥️ Como rodar

```bash
git clone https://github.com/MatheusJalles/Movi-menta
cd movimenta
# Abra o index.html no navegador
```

Não requer instalação de dependências — JavaScript puro.

---

## 💡 O que aprendi

- Manipulação avançada do DOM com JavaScript
- Organização e separação de responsabilidades no código
- Estrutura semântica de HTML
- Lógica de estado simples sem frameworks
